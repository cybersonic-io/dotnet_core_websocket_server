# .NET Core Websocket Server

A small .NET Core Websocket Server Testproject.

#### To start use:

```
// shows help
dotnet run -- /h

// start server @ Port 9000
dotnet run -- /p 9000
```

#### Build with following command
```
// standard build (dll)
dotnet build

// windows x64 build (websocket.exe)
dotnet build -r win10-x64
```

#### Restore Packages
```
dotnet restore
```
